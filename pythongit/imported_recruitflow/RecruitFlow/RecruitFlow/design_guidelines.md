# PSM Recruiting CRM Design Guidelines

## Design Approach
**System-Based Approach** using Material Design 3 principles adapted for PSM's premium enterprise recruiting platform. Prioritizes data density, professional aesthetics, and operational efficiency with PSM's corporate branding integration.

## Core Design Elements

### A. Color Palette
**Primary Brand Colors:**
- Primary: 207 91% 24% (PSM Dark Navy #1e293b)
- Accent: 199 89% 48% (PSM Bright Blue #0ea5e9)
- Surface Dark: 215 25% 11% (Deep navy backgrounds)
- Surface Light: 210 20% 96% (Clean light backgrounds)

**Supporting Colors:**
- Success: 142 71% 45% (Professional green)
- Warning: 38 92% 50% (Amber alerts)
- Error: 0 84% 60% (Clear red indicators)
- Text Primary: 215 20% 91% (High contrast on dark)
- Text Secondary: 215 16% 65% (Muted text)

### B. Typography
**Google Fonts:**
- Primary: Inter (400, 500, 600) - premium readability for data interfaces
- Secondary: JetBrains Mono (400) - technical data, IDs, timestamps

**Hierarchy:**
- Headers: 600 weight, strategic sizing
- Body: 400-500 weight, 14-16px optimal
- Data/Captions: 400 weight, 12-14px
- PSM Logo: Prominent placement with brand consistency

### C. Layout System
**Tailwind Spacing Units:** 2, 4, 6, 8
- Micro spacing: p-2, gap-2 (form inputs, tight layouts)
- Standard spacing: p-4, gap-4 (cards, components)
- Section spacing: p-6, gap-6 (content areas)
- Major spacing: p-8, gap-8 (page sections, headers)

### D. Component Library

**Navigation & Branding:**
- PSM logo prominently displayed in header/sidebar
- Dark navy sidebar with bright blue accent highlights
- Breadcrumb navigation with PSM color scheme
- Tab navigation using brand accent colors

**Data Management Components:**
- Professional data tables with PSM navy headers
- Candidate/client profile cards with subtle brand accents
- Recruitment pipeline visualizations
- Status indicators using PSM color palette
- Advanced filtering interfaces for bulk operations

**Premium UI Elements:**
- Elevated cards with subtle shadows on dark backgrounds
- Bright blue floating action buttons
- Professional modal dialogs with PSM branding
- Toast notifications matching brand colors
- Loading states with PSM accent colors

**Business-Critical Forms:**
- Clean form layouts with PSM navy groupings
- File upload areas for resumes/documents
- Rich text editors for professional correspondence
- Scheduling interfaces with brand-consistent styling

### E. Layout Patterns

**Executive Dashboard:**
- Premium widget layouts showcasing key recruiting metrics
- PSM-branded quick action areas
- Professional activity feeds with navy/blue theming

**Operational Views:**
- Advanced data tables for candidate/client management
- Sophisticated search with PSM-styled filters
- Bulk operation interfaces for efficiency
- Professional detail views with brand-consistent tabs

**Premium Features:**
- Full-screen document viewers
- Contextual information panels
- Professional confirmation dialogs
- Executive reporting interfaces

## PSM Brand Integration

**Visual Identity:**
- PSM logo strategically placed throughout interface
- Consistent use of corporate navy/blue palette
- Professional typography reflecting premium positioning
- Clean, modern aesthetic suitable for C-level presentations

**Professional Standards:**
- Enterprise-grade visual polish
- Consistent branding across all modules
- Premium color treatments for competitive differentiation
- Clean, trustworthy design language

## Interaction Principles

1. **Premium Efficiency:** Streamlined workflows for recruiting professionals
2. **Brand Consistency:** PSM colors and logo integration throughout
3. **Data Focus:** Clear information hierarchy with professional styling
4. **Enterprise Polish:** Sophisticated visual treatments
5. **Minimal Animation:** Subtle, professional transitions only

This design approach delivers a premium recruiting technology platform that reinforces PSM's professional brand while optimizing for data-intensive recruiting operations.
